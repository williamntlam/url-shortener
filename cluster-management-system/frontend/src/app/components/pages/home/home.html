What is ShortURL?

I made this URL Shortener app to practice my system design skills. In particular, I introduced myself to Kubernetes and made a cluster for the following: A Redis pod for caching URLs, a Key Generation Service (KGS), a Postgres Database, and a Frontend and Backend pod.

Key Points:

  • Containerized with Docker for consistent deployment
  • Orchestrated using Kubernetes for scalability
  • Redis for high-performance caching
  • Custom Key Generation Service (KGS) for unique URLs
  • PostgreSQL for reliable data persistence
  • Angular & FastAPI for modern web stack

Key Features:

  • Distributed URL shortening with high availability
  • Fast response times through Redis caching
  • Scalable architecture using Kubernetes
  • Modern UI with responsive design
  • RESTful API endpoints

Implementation Details:

Frontend

Built with:
  • Angular for strict component architecture
  • TailwindCSS for modern, responsive styling

Backend

Developed using FastAPI for:
  • High-performance API endpoints
  • Automatic API documentation
  • Async request handling

Redis Cache

Maintains two key sets with high availability:
  • Used keys for quick URL lookups
  • Unused keys pool that's periodically replenished
  • Each Redis instance has a replica for fault tolerance
  • Automatic failover if primary instance goes down

Key Generation Service (KGS)

Ensures unique URL generation with high availability through pod replication

PostgreSQL Database

Implemented as a StatefulSet for:
  • Persistent storage of URL mappings
  • Data replication for fault tolerance
  • Accurate state management
